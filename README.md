# Justin Perry Portfolio

**Live Site:** https://lilnewday.github.io/Updated-Portfolio/

## Description

This project is my personal developer portfolio website. It showcases my technical skills, featured projects, and professional background in software development.
The portfolio highlights several applications I have built using modern web technologies including Javascript, React, Node.js, and REST APIs. It serves as a central hub where employers and collaborators can explore my work, view live project demos, and access my resume and contact information.
The goal of this portfolio is to demonstrate my ability to design, develop, and present functional web applications while continuing to grow as a software developer.

## Technologies Used

- HTML
- CSS
- Javascript
- Bootstrap
- React
- Github Pages

## Installation

No installation is required to view this project.
You can access the live portfolio here:
https://lilnewday.github.io/Updated-Portfolio/

## Usage

This portfolio website allows visitors to:

- Learn more about my background and technical skills
- Explore featured software development projects
- View links to live applications and GitHub repositories
- Access my resume
- Connect with me through Github and LinkedIn
